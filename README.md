### 引用方式
在根目录的build里添加
```java
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
	}
}
  ```
  在需要引用的module的build里添加
  ```java
 dependencies {
	implementation 'com.github.ljwx:EditTextActive:Tag'
}
  ```
